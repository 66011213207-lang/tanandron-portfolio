# assets/gallery

วิธีอัพรูปแบบเร็วที่สุด (ไม่ต้องแก้โค้ดทีละรูป):

1. เลือกโฟลเดอร์หมวดที่ตรงกับรูป — แต่ละโฟลเดอร์ในนี้คือ 1 หมวด/แท็กกรองในหน้าเว็บ
   (มีให้แล้ว: `Activity`, `Game`, `Prop`, `Environment`, `Character`, `Other` (เช่นงาน App/UX/UI
   หรืองานอื่นที่ไม่เข้าหมวดไหน) — จะสร้างหมวดใหม่
   เพิ่มก็ได้ แค่สร้างโฟลเดอร์ใหม่ในนี้ ชื่อโฟลเดอร์ = ชื่อหมวดที่จะโชว์)
2. ลากไฟล์ .jpg / .jpeg / .png / .webp / .gif วางลงในโฟลเดอร์นั้นได้เลย ทีละกี่รูปก็ได้
   ตั้งชื่อไฟล์ให้สื่อความหมายหน่อย เช่น `photo-contest-2024.jpg` เพราะชื่อไฟล์จะถูกใช้
   เป็นคำบรรยายใต้รูปอัตโนมัติ (ขีดล่าง/ขีดกลางจะถูกแปลงเป็นช่องว่างให้)
3. ดับเบิลคลิก `update-gallery.bat` ที่ root โปรเจกต์ (หรือรัน
   `node scripts/build-gallery.js` ใน terminal) — สคริปต์จะสแกนโฟลเดอร์นี้ทั้งหมด
   แล้วเขียน `GALLERY_ITEMS` ใน `js/main.js` ให้ใหม่ทั้งก้อนโดยอัตโนมัติ
4. รีเฟรชหน้าเว็บ ครบ

ไม่ต้องแก้ `js/main.js` เอง — ส่วน `GALLERY_ITEMS` ถูกคุมโดยสคริปต์ทั้งหมด
